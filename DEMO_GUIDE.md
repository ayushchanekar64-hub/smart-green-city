### 📝 Report Issue - Complete Flow

**Step-by-step demo:**

1. **Navigate:** Click "Report Issue" in navbar

2. **Fill the form:**
   - **Issue Type:** Select "Garbage & Waste"
   - **Title:** "Overflowing garbage bin on Main Street"
   - **Description:** "The garbage bin near the park has been overflowing for 3 days. Creating hygiene issues."
   - **Location:** Click "📍 Get Current Location" (allow browser permission) OR type "Main Street, Downtown"
   - **Image:** Click "Choose File" and select any image

3. **Submit:** Click "Submit Complaint"

4. **Success screen:**
   - ✅ Green checkmark animation
   - Unique Complaint ID (e.g., CMP7X9K2L)
   - **IMPORTANT:** Copy this ID!

5. **Actions:**
   - Click "Report Another Issue" to submit more
   - Click "Track Complaint" to see status

**Demo tip:** Report 3-4 different issues to populate the dashboard and analytics!

---

### 📊 Track Complaint - Status Monitoring

**Demo flow:**

1. **Navigate:** Click "Track" in navbar

2. **Search:**
   - Paste your Complaint ID
   - Click "Track" button

3. **View details:**
   - Complaint information card (left)
   - Progress timeline (right)
   - Status badge (color-coded)

4. **Progress stages:**
   - 🟡 **Submitted** - Initial stage
   - 🔵 **Under Review** - Being evaluated
   - 🔵 **In Progress** - Being worked on
   - 🟢 **Resolved** - Completed

5. **Recent complaints:**
   - Scroll down to see recent submissions
   - Click any complaint to view instantly

**Demo tip:** Keep your Complaint ID handy to test admin updates!

---

### 🎯 Admin Dashboard - Management Interface

**Demo flow:**

1. **Navigate:** Click "Admin" in navbar

2. **Overview stats:**
   - Total complaints
   - Pending count
   - In Progress count
   - Resolved count

3. **Filter complaints:**
   - Click "All" to see everything
   - Click "Pending" to see new complaints
   - Click "In Progress" to see active work
   - Click "Resolved" to see completed

4. **Manage a complaint:**
   - Click any complaint row (it highlights)
   - Right panel shows full details
   - View all information

5. **Update status:**
   - Scroll to "Update Status" section
   - Click status buttons:
     - "Pending" → "In Progress"
     - "In Progress" → "Resolved"
   - Status updates instantly!

6. **Delete complaint:**
   - Scroll to bottom
   - Click red "Delete Complaint" button
   - Confirm deletion

**Demo tip:** Update a complaint status, then go to Track page to see the change!

### 💨 Air Quality Monitor - Real-time AQI

**Features:**

1. **Main AQI Display:**
   - Large number (52)
   - Status: Good/Moderate/Unhealthy
   - Color-coded gauge bar
   - Description of air quality

2. **AQI Scale:**
   - 0-50: Good (Green)
   - 51-100: Moderate (Yellow)
   - 101-150: Unhealthy for Sensitive (Orange)
   - 151-200: Unhealthy (Red)
   - 201-300: Very Unhealthy (Dark Red)
   - 300+: Hazardous (Maroon)

3. **Location-wise AQI:**
   - Downtown: 68
   - Industrial Area: 142
   - Residential Zone: 45
   - Park Area: 32
   - Highway Junction: 89
   - Suburban Area: 41

4. **Pollutant Levels:**
   - PM2.5, PM10, O3, NO2, SO2, CO
   - Each with status badge

5. **Health Recommendations:**
   - Based on current AQI
   - Actionable advice

6. **7-Day Forecast:**
   - Visual bar chart
   - Daily AQI predictions

**Demo tip:** Check different locations to compare air quality!
